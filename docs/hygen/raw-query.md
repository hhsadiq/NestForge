# Raw queries

---

This guide explains how to generate and organize raw SQL queries when needed.

## Usage

```bash
npm run generate:raw-query
```

Follow the prompts to select the target module and define the query file.

## Location

Queries are placed under the module’s relational persistence layer in a `queries` folder.
