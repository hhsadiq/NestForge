# Version

---

This page explains the built-in versioning helpers for generators.

## Overview

We maintain versioned templates and helper scripts for:

- Domain versions
- Entities versions
- Mappers versions

See `scripts/versioning/*.js` for details.

## Usage

Versioning is applied automatically by the generators. You can also run the scripts directly if needed.
